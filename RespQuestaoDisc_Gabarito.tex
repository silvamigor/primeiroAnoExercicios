Seja \begingroup \mathcode`\:="003A $f: \;\bbR \to \bbR$ \endgroup uma função tal que \(f(1) = \frac{2}{3}\) e que \(f(x+y) = f(x)+f(y)\), para todo \(x, y \in \bbR\). Vamos determinar o valor de \(f(3)\). Para isso observe que 
%
\[
    f(3) = f(2 + 1) = f(2) + f(1) = f(1 + 1) + f(1) = f(1) + f(1) + f(1) = \dfrac{2}{3} + \dfrac{2}{3} + \dfrac{2}{3} = 2.
\]
%

\vspace{8pt}
Portanto, \(f(3) = 2\).
}
