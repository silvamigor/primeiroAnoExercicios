\begin{tikzpicture}[remember picture, overlay]
\node[
  anchor=north west,
  xshift=3.2cm,
  yshift=-1.7cm,
  text width=17cm,
  align=left
] at (current page.north west)
{
\cursive{
    \setlength{\baselineskip}{0.7cm}
    %\begingroup \mathcode`\:="003A
    
    Vamos determinar o lucro obtido ao vender \(800\) ingressos para um evento. O lucro em função do número de ingressos vendidos é determinado pela fórmula
    \vspace{-3pt}
    \[
        L(x) = A(x) - C(x),
    \]
    
    \vspace{-6pt}
    onde \(A(x)\) é a arrecadação, \(C(x)\) é o custo operacional e \(x\) é o número de ingressos vendidos. As funções \(A(x)\) e \(C(x)\) serão determinadas a partir das informações extraídas do gráfico. Para isso, note que \(A(0) = 0\) e \(A(600) = 6600\). Assim, seu coeficiente linear é zero e o coeficiente angular é igual a taxa de variação média:
    \[
        \dfrac{6600 - 0}{600 - 0} = 11.
    \]
    
    \vspace{5pt}
    Logo,
    \[
        A(x) = 11x.
    \]
    
    \vspace{10pt}
    Para \(C(x)\), temos que \(C(0) = 2400\) e \(C(600) = 6600\). Nesse caso, o coeficiente linear, dado pela interseção do gráfico com o eixo das coordenadas, é \(2400\) e o coeficiente angular é a taxa de variação média:
    \[
        \dfrac{6600 - 2400}{600 - 0} = \dfrac{4200}{600} = 7.
    \]
    
    \vspace{10pt}
    Assim, \(C(x) = 7x + 2400\). Portanto,
    \[
        L(x) = A(x) - C(x) = 11x - 7x - 2400 = 4x - 2400.
    \]
    
    \vspace{-5pt}
    Finalmente, lucro obtido ao vender \(800\) ingressos é
    \[
        L(800) = 4\cdot 800 - 2400 = 800 \, \text{ reais}.
    \]
    }
};
\end{tikzpicture}
